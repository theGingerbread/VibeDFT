# calculator/qe

Canonical QE backend boundary for the VibeDFT v2 skeleton.

Every task subpackage owns:

- `params.py`
- `prepare.py`
- `run.py`
- `monitor.py`
- `parse.py`
- `clean.py`

These files are placeholders only. No legacy QE implementation is treated as
canonical until it is explicitly re-expressed inside this layout.
